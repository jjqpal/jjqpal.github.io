Add Two

题目描述

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,

return [0, 1].

添加已经存在的工程

使用原工程自带的Makefile

  1. “File -> New -> Project”，选择“Empty Project”或者“console application“并创建。
  2. 选中 Project，右键，选择“Add Files Recursively”，将整个项目文件夹加入工程。
  3. 选中 Project，右键，选择“Properties”，“Project settings”选项卡中的“Makefile”填入Makefile 文件名，并勾选“This is a custom Makefile”。
  4. 选中 Project，右键，选择“Build options”，删除“”Make” command”选项卡中所有的“$target”。
  5. 修改已有的Makefile，使输出路径为codeblocks默认路径，否则会一直提示你没有build。

使用自动生成的Makefile

  1. “File” -> “New” -> “Project”，选择“Empty Project”或者“console application“并创建。
  2. 选中 Project，右键，选择“Add Files Recursively”，将整个项目文件夹加入工程。
  3. 选中 Project，右键，选择“Build options”，进入Linker Settings子目录，将项目所需要的lib库添加进Link libraries中，有时lib库的顺序不对也会影响编译，会发生“未定义的引用”错误。
  4. “Build options”->“Search Directories”->”Compiler” 添加头文件目录。
  5. “Build options”->“Search Directories”->”Linker” 添加自己的lib文件的目录，这个也要注意顺序，否则会发生“未定义的引用”错误。
